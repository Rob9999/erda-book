# ERDA — Startwelle Grafiken (Draft SVG)

Diese Ablage enthält eine **erste direkt nutzbare Grafikserie** als SVG-Entwürfe für den argumentativen Backbone des ERDA-Buchs.

## Status

- **Format:** SVG
- **Stand:** redaktionelle Entwurfsfassungen
- **Ziel:** schnelle Sichtbarmachung zentraler Architektur- und Governance-Logiken
- **Weiterverarbeitung:** SVGs können in Inkscape, Illustrator oder Figma nachbearbeitet werden

## Arbeitsnotizen

1. Die SVGs sind bewusst **ruhig, sachlich und staatsmännisch** gehalten.
2. Text und Mikrotypografie sollten vor Buchintegration noch feinredigiert werden.
3. Für DE/EN-Doppelnutzung empfiehlt sich eine zweite Sprachfassung oder ein textarmer Export.
4. Die zugrunde liegenden kontextuellen Prompts stehen in [../erda-visual-prompts.md](../erda-visual-prompts.md).
