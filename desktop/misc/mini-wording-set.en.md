# Mini Wording Set (canonical)

This set is an editorial reference to keep key terms consistent throughout the book.

## Weaponized Interdependence

**Core formula:** Dependence becomes leverage.

**Mnemonic:**  
Interdependence is not a guarantee of peace when not all partners are allies.

**Standard sentence (short):**  
Trade policy becomes security architecture once dependencies can be weaponised.

## Pragmatic Federalism

**Core formula:** Opt‑in capacity to act now, federal direction visible, democratically controlled.

**Standard sentence (short):**  
Better capable with the willing than blocked with everyone – with an open door and anti‑sabotage safeguards.

## Unity-through-Action

**Core formula:** Unity emerges through joint action.

**Standard sentence (short):**  
Not “unity first, action later”, but: action creates unity.
