## 3) Beitrittsbedingungen: klar, messbar, „kein Veto über Beitritte“

### A) Politisch-rechtliche Mindestkriterien (Rule-of-law / Demokratie)

| Kriterium                                  | Messbar als (Beispiel)                              | Go      | NoGo                                  |
| ------------------------------------------ | --------------------------------------------------- | ------- | ------------------------------------- |
| Freie, faire Wahlen + Machtwechsel möglich | unabhängige Wahlaufsicht, OSZE-Standards            | erfüllt | systematische Wahlmanipulation        |
| Unabhängige Gerichte                       | Richterernennung abgesichert, Urteile vollstreckbar | erfüllt | politische Kontrolle über Justiz      |
| Medienfreiheit + Pluralismus               | Schutz vor staatlicher Übernahme                    | erfüllt | staatliche Mediengleichschaltung      |
| Anti-Korruption / Transparenz              | Lobby-/Parteifinanzregeln, Register                 | erfüllt | „state capture“-Risiko                |
| Grundrechte bindend                        | Charta + Verfassungsgerichtsbarkeit                 | erfüllt | Grundrechtsrelativierung per Notstand |

### B) Sicherheits- und Verteidigungsfähigkeit (Pay-to-play)

| Kriterium              | Mindeststandard (Beispiel)                                       | Go                   | NoGo                      |
| ---------------------- | ---------------------------------------------------------------- | -------------------- | ------------------------- |
| Verteidigungsbeitrag   | fester Anteil am BIP **oder** Fähigkeitsbeitrag (Truppen/Assets) | planbar, verbindlich | „Absicht“ ohne Vertrag    |
| Interoperabilität      | gemeinsame Standards, Datenlinks, Munition, Ausbildung           | zertifiziert         | nationale Sonderwege      |
| Sicherheitsüberprüfung | Schutz vor Infiltration, Geheimschutz                            | kompatibel           | Leaks/Einflussoperationen |
| Einsatz-/Kontrollkette | parlamentarische Zustimmung + definierte Notstandsregeln         | geregelt             | Exekutive ohne Kontrolle  |

### C) Wirtschaft/Industrie/Resilienz (Anti-Erpressbarkeit)

| Kriterium                       | Minimum                                                  | Go          | NoGo                                |
| ------------------------------- | -------------------------------------------------------- | ----------- | ----------------------------------- |
| Sanktionskonformität            | Vollzug, Anti-Evasion, Exportkontrolle                   | nachweisbar | systematische Umgehung              |
| Energie-/Infrastrukturresilienz | Notfallpläne, Redundanz, Schutz kritischer Infrastruktur | auditiert   | Single-point-of-failure             |
| Beschaffungsdisziplin           | gemeinsame Spezifikationen, Audit, Korruptionsschutz     | kompatibel  | „Buy national“ ohne Ausnahme-Regeln |

## 4) Beitrittsmechanik: so bleibt es evolutiv stabil

| Mechanismus                             | Sinn                                   | Umsetzung                                                                                    |
| --------------------------------------- | -------------------------------------- | -------------------------------------------------------------------------------------------- |
| **Kein Einstimmigkeitszwang**           | verhindert Erpressung                  | Beitritt via **qualifizierte Mehrheit** im Club-Parlament + 2. Kammer                        |
| **Stufenbeitritt (Tiers)**              | niemand muss alles sofort              | Tier 0 Beobachter → Tier 1 Beschaffung/Standards → Tier 2 operativ → Tier 3 volle Föderation |
| **Konditionalität**                     | schützt vor innerer Erosion            | Gelder/Programme nur bei Rule-of-law-Compliance                                              |
| **Safeguard-Klauseln**                  | verhindert „Capture“                   | Stimmrechts-/Mittel-Suspendierung bei schweren Verstößen (gerichtsfest)                      |
| **Exit-Regel + geordnete Entflechtung** | reduziert Angst, erhöht Freiwilligkeit | klare Fristen, Abwicklungsregeln, Schutz von Minderheitenrechten                             |
